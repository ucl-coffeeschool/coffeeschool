##Goal: Repeating code

Another commonly used construct is a loop, to repeat code multiple times. For example:

```javascript
for (var i = 0; i < 10; i++) {
  console.log(i);
}
```
will output the numbers 0 to 9.

Now write a loop to output the numbers 0 to 9 multiplied by 2, ie 0, 2, 4...
